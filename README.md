# Graduation-project
Моя часть командного дипломного проекта.
Проект: тестирование учебного сайта интернет-магазина https://demo1wp.shoporg.ru/.
ТЗ: отсутствует.
Мои действия: 
- разработка и документирование юзер-стори;
- визуализация user story mapping;
- составление 70 % чек-листа;
- проведение нагрузочного тестирования (Apache JMeter) и составление отчета;
- проведение СЕО-тестирования;
- проведение функционального тестирования;
- заведение тест-кейсов и прогонов в TestRail;
- проектирование автотестов на Python;
- кросс-браузерное тестирование;
- подготовка итоговой презентации.
